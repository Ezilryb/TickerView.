Quels marchés ?
Crypto Futures (Binance, OKX, Bybit) · Forex (FXCM) · Futures CME (NQ, ES, CL) — AAPL, TSLA et indices US en bêta Q3 2026.
Quelle source de données ?
Stream AggTrades WebSocket Binance directement dans le navigateur. Latence médiane mesurée : <50ms. Pas d'intermédiaire, pas de polling.
Pourquoi pas TradingView ?
TradingView n'a pas de Footprint Chart natif, ni de delta tick-par-tick, ni de VRVP temps réel sans plugin tiers payant. TickerView intègre tout nativement.
Gratuit ou payant ?
Beta publique gratuite, sans inscription obligatoire. Les plans Pro (multi-exchange, alertes, API export) arriveront post-beta.
Web ou desktop ?
Web app pure — Chrome, Firefox, Edge. Installable comme PWA. Aucune installation, aucun téléchargement. Fonctionne sur Mac, Windows, Linux.
